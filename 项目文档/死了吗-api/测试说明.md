## 1. 测试签到功能

在 **Mac 终端** 执行（复制整行，回车）：

```bash
curl -X POST https://qxzprrbklqtx.usw-1.sealos.app/api/check-in \
  -H "Content-Type: application/json" \
  -d '{"code":"test_code","nickname":"测试用户","email":"1248922132@qq.com"}'
```

**说明**：`code: "test_code"` 为测试模式，跳过微信校验。正常返回类似：
```json
{"code":0,"message":"签到成功","data":{"checkInDate":"2026-01-30"}}
```

## 2. 测试邮件发送功能

在 **Mac 终端** 执行（把邮箱改成你的）：

```bash
curl -X POST https://qxzprrbklqtx.usw-1.sealos.app/api/test/send-email \
  -H "Content-Type: application/json" \
  -d '{"to":"1248922132@qq.com"}'
```

**说明**：会向该邮箱发送一封测试邮件，标题和正文与提醒邮件一致。正常返回：
```json
{"code":0,"message":"邮件已发送","data":{"to":"1248922132@qq.com"}}
```
